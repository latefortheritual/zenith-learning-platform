# Zenith

> Reaching the highest point in knowledge.

Zenith is a full-stack adaptive learning platform built as a Bachelor's thesis project at the **Faculty of Cybernetics, Statistics and Economic Informatics**.

## Tech Stack
- **Backend:** Spring Boot (Java)
- **Frontend:** React
- **Database:** PostgreSQL
- **Algorithms:** IRT 1PL (Rasch model), Bayesian Knowledge Tracing (BKT)
